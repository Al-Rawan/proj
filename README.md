# قاعة الزاب الرياضية - Al-Zab Sports Hall

مجموعة من جداول التمارين الرياضية المنظمة لشهر البناء والرشاقة.

A collection of organized workout schedules for building and agility phases.

## محتويات المشروع / Project Contents

- **جدول البناء (الشهر الأول)**: خطة تدريبية مركزة لبناء العضلات والقوة.
- **جدول الرشاقة**: خطة تدريبية تهدف إلى تحسين اللياقة البدنية والتحمل.
- **Building Table (Month 1)**: A focused training plan for building muscle and strength.
- **Agility Table**: A training plan aimed at improving fitness and endurance.

## التقنيات المستخدمة / Technologies Used

- HTML5 / CSS3 / JavaScript
- Google Fonts (Cairo)
- LocalStorage API (لحفظ البيانات)

## 📋 المميزات الجديدة (v1.0) / New Features
*   **نظام الأيام الديناميكي:** يمكنك الآن إضافة يوم رابع، خامس، أو حتى سابع للجدول، أو حذف الأيام غير المرغوب فيها.
*   **تعديل حي:** يمكنك تعديل أسماء التمارين والجولات مباشرة من واجهة "التعديل".
*   **الحفظ التلقائي:** يتم حفظ جميع التعديلات في ذاكرة المتصفح (LocalStorage) وتظل موجودة حتى بعد إغلاق الملف.
*   **النسخ الاحتياطي:** تمت إضافة أزرار لتصدير واستيراد البيانات كملفات JSON لضمان عدم ضياع مجهودك عند تغيير المتصفح.

## 🛠️ كيفية الاستخدام / How to Use
1. افتح الملف (`جدول البناء.html` أو `جدول الرشاقة.html`) في المتصفح.
2. اضغط على زر **"✏️ تعديل"** لفتح لوحة التحكم.
3. أضف الأيام أو التمارين كما تشاء، ثم اضغط **"💾 حفظ كافة التغييرات"**.
4. اضغط على زر **"🖨️ طباعة"** للحصول على نسخة ورقية احترافية.
